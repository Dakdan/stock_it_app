## 🎯 ฟีเจอร์หลัก

- 📋 จัดการงาน Tickets
- 👥 มอบหมายงานให้เจ้าหน้าที่ IT
- 📊 Dashboard สรุปผลงาน
- 🔗 เชื่อมต่อ Google Sheet
